## 🧠 About Me

I'm an AI/ML Engineer doing my M.Sc. in Digital Technology at **Ostfalia Hochschule**, Germany. Most of my work sits at the intersection of multi-agent systems, LLM fine-tuning, and getting ML pipelines into production.

Some things I've built so far:

- 🤖 **Multi-agent AI museum guide** on Temi robots with RAG-based retrieval (WOBCOM)
- 🚗 **Constrained LLM fine-tuning** for automated test generation (Volkswagen AG)
- 🏭 **Text2CAD** pipeline that turns plain text into manufacturable 3D CAD models using a multi-agent system
- 📊 **ML-based ticket sales forecasting** for a professional ice hockey team
- 🤖 **ROS 2 robotics projects** with camera-based navigation and Sokoban puzzle solving
- 🚙 **Autonomous driving** mini-projects: deep reinforcement learning, sensor fusion, adaptive cruise control

I like taking ideas from research papers and making them actually work. If it's running in production, even better.
